# Course-Registration-System
A simple console based Course Registration System written in C++

---

## Features

- Registers new Students and allows them to enroll in multiple courses
- Existing Students can choose to Register for more courses
- Students can choose to Drop existing courses
- Students can view details of their profile and courses

---

## Key Concepts

- Encapsulation
- Constructors
- Vectors
- Linear Search

---

## How to Run

### Requirements
- C++ compiler (g++ or MSVC)

### Compile & Run (g++)

```bash
g++ main.cpp source.cpp -o CourseRegSystem
./CourseRegSystem
```
